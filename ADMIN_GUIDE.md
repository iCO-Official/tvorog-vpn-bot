# 🧀 Шпаргалка админа Творог VPN

## Основные команды

| Команда | Описание |
|---------|----------|
| `/admin` | Открыть админ-панель |
| `/add_user USER_ID DAYS` | Добавить подписку |
| `/cheese_orders` | Список pending заказов творога |
| `/order_cheese USER_ID` | Открыть Ozon для заказа |
| `/set_cheese USER_ID STATUS` | Изменить статус творога |

## Статусы творога

| Статус | Описание |
|--------|----------|
| `pending` | Ожидает адреса |
| `ordered` | Заказан на Ozon |
| `delivered` | Доставлен |
| `none` | Сбросить |

## Примеры использования

### Добавить подписку на 30 дней
```
/add_user 123456789 30
```

### Посмотреть pending заказы
```
/cheese_orders
```

### Открыть Ozon для заказа творога
```
/order_cheese 123456789
```

### Отметить что творог заказан
```
/set_cheese 123456789 ordered
```

### Отметить что творог доставлен
```
/set_cheese 123456789 delivered
```

### Сбросить статус творога
```
/set_cheese 123456789 none
```

## Процесс заказа творога

1. Пользователь покупает подписку от 1 месяца
2. Пользователь нажимает "Получить творог"
3. Бот показывает список городов
4. Пользователь выбирает город
5. Бот показывает пункты выдачи Ozon
6. Пользователь выбирает пункт
7. Бот отправляет тебе сообщение
8. Ты заказываешь творог на Ozon
9. Помечаешь: `/set_cheese USER_ID ordered`
10. После доставки: `/set_cheese USER_ID delivered`
