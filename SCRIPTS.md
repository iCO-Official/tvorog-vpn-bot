# Скрипты VPN-бота

## Основные скрипты

### Запуск и остановка

| Скрипт | Описание |
|--------|----------|
| `start_bot.sh` | Запуск бота (Linux) |
| `start_bot.bat` | Запуск бота (Windows) |
| `stop_bot.sh` | Остановка бота |
| `restart_bot.sh` | Перезапуск бота |

### Управление пользователями

| Скрипт | Описание |
|--------|----------|
| `add_user.py` | Добавление пользователя |
| `remove_user.py` | Удаление пользователя |

### Мониторинг

| Скрипт | Описание |
|--------|----------|
| `status.sh` | Проверка статуса |
| `monitor.sh` | Мониторинг в реальном времени |
| `logs.sh` | Просмотр логов |

### Обслуживание

| Скрипт | Описание |
|--------|----------|
| `backup.sh` | Резервное копирование |
| `restore.sh` | Восстановление из бэкапа |
| `update_bot.sh` | Обновление бота |
| `cleanup.sh` | Очистка старых файлов |
| `uninstall.sh` | Удаление бота |

### Установка

| Скрипт | Описание |
|--------|----------|
| `setup_server.sh` | Автоматическая настройка сервера |
| `install_deps.sh` | Установка зависимостей |

### Тестирование

| Скрипт | Описание |
|--------|----------|
| `test_bot.py` | Тестирование компонентов |
| `test_local.py` | Локальное тестирование |

### Утилиты

| Скрипт | Описание |
|--------|----------|
| `qr_generator.py` | Генерация QR-кодов |
| `generate_report.py` | Генерация отчётов |
| `generate_sample_config.py` | Генерация примера конфигурации |
| `check_server.sh` | Проверка состояния сервера |

## Использование

### Первый запуск

```bash
# 1. Установка зависимостей
chmod +x install_deps.sh
./install_deps.sh

# 2. Настройка конфигурации
cp config.py.example config.py
nano config.py

# 3. Настройка сервера
chmod +x setup_server.sh
sudo ./setup_server.sh

# 4. Запуск бота
chmod +x start_bot.sh
./start_bot.sh
```

### Ежедневное обслуживание

```bash
# Проверка статуса
./status.sh

# Просмотр логов
./logs.sh

# Резервное копирование
./backup.sh
```

### Управление пользователями

```bash
# Добавление пользователя
python3 add_user.py USER_ID 30

# Удаление пользователя
python3 remove_user.py USER_ID

# Генерация отчёта
python3 generate_report.py
```

## Примеры

```bash

```

### Резервное копирование перед обновлением


### Очистка старых файлов

```bash

```
