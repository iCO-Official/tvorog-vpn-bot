# Пример использования VPN-бота

## Сценарий 1: Новый пользователь

1. Пользователь находит бота в Telegram
2. Отправляет `/start`
3. Бот приветствует и показывает меню
4. Пользователь нажимает "💰 Купить VPN"
5. Выбирает тариф (например, "1 месяц — 299 Stars")
6. Оплачивает через Telegram Stars
7. Получает конфигурационный файл
8. Устанавливает WireGuard на iPhone
9. Импортирует конфиг
10. Наслаждается безопасным интернетом!

## Сценарий 2: Продление подписки

1. Пользователь отправляет `/status`
2. Видит, что подписка истекает завтра
3. Нажимает "💰 Купить VPN"
4. Покупает новый тариф
5. Подписка продлевается

## Сценарий 3: Пробный период

1. Новый пользователь отправляет `/start`
2. Бот автоматически даёт 3 дня бесплатно
3. Пользователь пробует VPN
4. Если нравится — покупает подписку

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню, регистрация |
| `/buy` | Купить подписку |
| `/status` | Проверить статус подписки |
| `/help` | Справка по использованию |
| `/admin` | Админ-панель (только для админа) |

## Оплата

Бот принимает оплату через **Telegram Stars** — встроенную валюту Telegram.

Преимущества:
- Без комиссий для пользователя
- Мгновенное зачисление
- Не нужен банковский счёт
- Анонимность

## Тарифы

| Тариф | Цена | Срок |
|-------|------|------|
| 1 неделя | 99 Stars | 7 дней |
| 1 месяц | 299 Stars | 30 дней |
| 3 месяца | 699 Stars | 90 дней |

## Установка VPN на устройства

### iPhone/iPad
1. Скачайте WireGuard из App Store
2. Откройте бота в Telegram
3. Нажмите "📥 Получить конфиг VPN"
4. Скачайте конфигурационный файл
5. Откройте файл в WireGuard
6. Нажмите на подключение

### Android
1. Скачайте WireGuard из Google Play
2. Следуйте инструкциям выше

### Windows/Mac
1. Скачайте WireGuard с wireguard.com
2. Установите приложение
3. Импортируйте конфиг

## Безопасность

- Все данные зашифрованы протоколом WireGuard
- Мы не храним логи активности
- Ключи обновляются автоматически
- Каждый пользователь получает уникальный ключ

## Масштабирование

### Добавление новых серверов
1. Арендуйте новый VPS
2. Установите WireGuard
3. Добавьте в конфигурацию бота
4. Предложите пользователям выбор серверов

### Реферальная программа
1. Каждый пользователь получает уникальную ссылку
2. Приглашённые друзья получают +7 дней бесплатно
3. Приглашающий получает +3 дня за каждого друга

## Мониторинг

### Просмотр статистики
```bash
# В боте
/admin

# На сервере
wg show
journalctl -u vpn-bot -f
```

### Проверка подключений
```bash
# Смотреть активные подключения
wg show wg0 latest-handshakes

# Смотреть трафик
wg show wg0 transfer
```

## Обслуживание

### Обновление бота
```bash
# На сервере
cd /home/vpnbot/vpn-bot
git pull
sudo systemctl restart vpn-bot
```

### Резервное копирование
```bash
# База данных
cp vpn_users.db vpn_users.db.backup

# Конфигурация
cp config.py config.py.backup
```

### Обновление ключей
Ключи обновляются автоматически при покупке новой подписки.
