# 🧀 Вклад в разработку Творог VPN

Спасибо за интерес к проекту! Мы приветствуем вклад от сообщества.

## Как внести вклад

### 1. Найдите проблему

- Просмотрите [issues](https://github.com/your-repo/tvorog-vpn-bot/issues)
- Создайте новую issue, если подходящей нет

### 2. Форкните репозиторий

```bash
git clone https://github.com/your-username/tvorog-vpn-bot.git
cd tvorog-vpn-bot
```

### 3. Создайте ветку

```bash
git checkout -b feature/your-feature-name
```

### 4. Внесите изменения

- Следуйте стилю кода проекта
- Добавьте тесты для нового функционала
- Обновите документацию

### 5. Протестируйте

```bash
python3 test_bot.py
```

### 6. Отправьте pull request

```bash
git add .
git commit -m "Add your feature"
git push origin feature/your-feature-name
```

## Стандарты кода

### Python

- Используйте PEP 8
- Добавляйте docstrings
- Используйте type hints
- Пишите тесты

### Коммиты

- Используйтеcriptive commit messages
- Формат: `type: description`
- Типы: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`

### Примеры

```
feat: add referral program
fix: resolve payment issue
docs: update README
style: format code
refactor: optimize database queries
test: add unit tests
chore: update dependencies
```

## Процесс

1. Fork → Branch → Commit → PR
2. Ожидайте ревью
3. Внесите исправления
4. Смерджите
