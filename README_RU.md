# 🧀 Творог VPN

Telegram-бот для продажи VPN-доступа через WireGuard. Работает с приложениями **Happ** (iPhone/Android) и **MantaRay** (Windows).

## Что умеет бот

- Продаёт VPN-ключи автоматически
- Принимает оплату через Telegram Stars
- Генерирует QR-коды для быстрого подключения
- Управляет пользователями
- Показывает статистику
- Выбор серверов

## Быстрый старт

### 1. Получите VPS сервер

Рекомендации:
- **Timeweb** — от 99₽/мес
- **Selectel** — от 300₽/мес
- **Hetzner** — от €3.5/мес

### 2. Получите Telegram Bot Token

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте `/newbot`
4. Введите имя: `Творог VPN Bot`
5. Введите username: `Tvorog_vpn_bot`
6. Скопируйте токен

### 3. Установите бота на сервер

```bash
# Подключитесь к серверу
ssh root@your-server-ip

# Скачайте бота
cd /home
git clone https://github.com/your-repo/tvorog-vpn-bot.git
cd tvorog-vpn-bot

# Запустите скрипт установки
chmod +x setup_server.sh
sudo ./setup_server.sh
```

### 4. Настройте конфигурацию

```bash
nano config.py
```

Замените:
- `YOUR_BOT_TOKEN` — ваш токен от BotFather
- `YOUR_SERVER_IP` — IP вашего сервера
- `ADMIN_ID` — ваш Telegram ID

### 5. Запустите бота

```bash
sudo systemctl start tvorog-vpn-bot
```

## Как пользоваться

### Для пользователей

1. Найдите бота в Telegram: @Tvorog_vpn_bot
2. Отправьте `/start`
3. Нажмите "Купить VPN"
4. Выберите тариф
5. Оплатите через Telegram Stars
6. Скачайте конфиг
7. Установите Happ (iPhone/Android) или MantaRay (Windows)
8. Импортируйте конфиг
9. Наслаждайтесь безопасным интернетом!

### Для администратора

```bash
# Просмотр статуса
sudo systemctl status tvorog-vpn-bot

# Просмотр логов
journalctl -u tvorog-vpn-bot -f

# Добавление пользователя
python3 add_user.py USER_ID 30

# Удаление пользователя
python3 remove_user.py USER_ID

# Резервное копирование
./backup.sh

# Восстановление
./restore.sh

# Мониторинг
./monitor.sh
```

## Установка VPN на устройства

### iPhone/Android (Happ)

1. Скачайте Happ из App Store / Google Play
2. Откройте бота в Telegram
3. Получите конфиг
4. Откройте Happ
5. Нажмите "+" → "Импорт конфига"
6. Выберите скачанный файл
7. Нажмите на подключение

### Windows (MantaRay)

1. Скачайте MantaRay с официального сайта
2. Установите приложение
3. Импортируйте конфиг
4. Активируйте подключение

## Настройка маршрутизации

В Happ/MantaRay вы можете выбрать:

- **Весь трафик** — всё через VPN
- **Определённые приложения** — только выбранные apps
- **Исключения** — определённые сайты в обход VPN
